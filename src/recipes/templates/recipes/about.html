About Recipe App

Recipe App is a project developed as part of the Python Course "Achievement 2". It allows users to browse, search, and filter recipes by various criteria.

Features

  • Browse all recipes
  • Filter by category
  • Search by name or ingredients
  • Filter by difficulty level
  • View detailed recipe information

Technologies Used

  • Python
  • Django
  • HTML/CSS
  • SQLite Database
